# Base Skill

## Purpose

定义当前仓库内所有 agent 的通用语言、风格和安全边界。

## Scope

适用于：

- 所有代码、文档、配置改动
- 所有分析、解释、handoff 输出

不适用于：

- 用户明确要求的外部系统人工操作

## Must

- 默认使用中文表达，必要时保留英文关键词、API 字段名和代码标识符
- 表达必须简洁、明确、可执行
- 结论必须基于仓库内可见事实或明确写出的假设

## Must Not

- 不臆测外部系统实时状态
- 不手工修改 `docs/` 下由 Swagger 生成的文件
- 不在没有需求的情况下创建新模块
- 不私自更改线上集群或其他外部系统

## Outputs

- 可执行结论
- 与仓库事实一致的说明

## Pass/Fail

- `Pass`：输出基于事实、边界清晰、语言统一
- `Fail`：出现臆测、越权操作或无依据结论
