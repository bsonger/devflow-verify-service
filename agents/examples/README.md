# Examples Reference

## Purpose

提供已经填好的 Harness 工件示例，作为模板实例而不是散乱笔记。

## Scope

适用于：

- 学习 `product spec -> sprint contract -> evaluator report -> handoff` 的完整写法
- 给新需求挑选最接近的参考案例

不适用于：

- 直接代替当前需求的真实工件

## Must

- 示例必须体现完整的 Harness 工件链路
- 使用示例时，应先理解结构，再回到 `templates/` 生成当前任务工件
- 当前保留的示例包括：
  - `release-status-alignment/`
  - `metadata-layer-split/`

## Must Not

- 不把示例内容直接复制为当前需求结论
- 不让示例替代当前任务的真实 run 记录

## Outputs

- 可参考的完整工件样例

## Pass/Fail

- `Pass`：示例能帮助新任务快速落地标准工件
- `Fail`：示例残缺、与模板脱节或无法复用
